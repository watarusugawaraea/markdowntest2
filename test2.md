# マークダウン記法 練習ファイル
このファイルでは、**Markdown（マークダウン）** の基本的な書き方を練習します。
---
## 1. 見出し
`#` の数で見出しレベルを変えられます。
```markdown
# 見出し1
## 見出し2
### 見出し3
```
---
## 2. 強調
* **太字**： `**太字**`
* *斜体*： `*斜体*`
* ~~取り消し線~~： `~~取り消し線~~`
---
## 3. リスト
### 箇条書き（順序なし）
```markdown
- りんご
- みかん
- 温州みかん
- 夏みかん
```
* りんご
* みかん
* 温州みかん
* 夏みかん
### 番号付きリスト
```markdown
1. はじめに
2. 実行手順
3. まとめ
```
1. はじめに
2. 実行手順
3. まとめ
---
## 4. 引用
```markdown
> これは引用です。
>> ネストした引用もできます。
```
> これは引用です。
>
> > ネストした引用もできます。
---
## 5. コード
### インラインコード
バッククォートで囲みます。
`print("Hello World")`
### コードブロック
```markdown
```
print("Hello Markdown!")
```
```
```python
print("Hello Markdown!")
```
